# This is the repository of our project

### Motivation

現在的媒體產業，其商業模式乃是透過網路瀏覽量轉換為廣告收益。

故，每篇文章的瀏覽量與其創造之收益息息相關。

而Facebook的讚數代表該文章於網路擴散的可能性。

本計劃嘗試透過機器學習方式，將既有資料餵入Spark，製作出預測文章讚數的應用
